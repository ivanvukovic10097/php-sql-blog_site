Popis fileova:
		img - dokument(direktoriji) u koji se slike spremaju
		Sopitas - dokument(direktorji) koji pokazuje izgled naslovnice i članka
		administracija.php - služi za upravljanje podatka članka
		clanak.php - prikazuje pojednični članak kad se klikne na njega unutar stranice
		connection.php - povezuje sve php dokumente sa bazom, koja se zove vijesti
		index.php - početna stranica
		kategorija.php - prikaz svih članaka vezanih za pojedinu kategoriju: Glazba(Music), Sport
		logout.php - odjava sa stranice administracija.php
		registracija.php - služi za kreiranje računa koji nema administratorska prava, razina 0
		skripta.php - služi za unos podataka u bazu preko unos.php dokumenta
		style.css - css file za sve dokumente
		unos.php - unos novosti ili članka
		vijesti.sql - baza podataka
